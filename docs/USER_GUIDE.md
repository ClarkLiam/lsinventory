# LS-Inventory User Guide

## Login
1. Open the application URL.
2. Sign in with your account credentials.

## Add a device
1. Go to **Inventory**.
2. Add INV code (`INV-XXXX`), unit ID (`XX`), and optional field (`XX`).
3. Save the device; default location is **Storage**.

## Prepare a project
1. Go to **Projects** and create a job/event.
2. Assign required devices.
3. Generate a packsheet.

## Packing workflow
1. Open **Packing** on mobile.
2. Scan QR codes while picking equipment.
3. Confirm all items are checked off.

## Maintenance
1. Open **Maintenance**.
2. Log service/damage details.
3. Track next service due dates.
